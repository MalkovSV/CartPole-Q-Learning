# Q‑Learning для CartPole‑v1

Реализация алгоритма Q‑Learning для решения задачи CartPole‑v1 из библиотеки Gymnasium.

## Описание проекта

Проект реализует обучение агента методом Q‑Learning для управления системой «тележка‑маятник» (CartPole). Агент учится удерживать маятник в вертикальном положении, перемещая тележку влево/вправо.

## Особенности реализации

- Дискретизация непрерывного пространства состояний
- ε‑жадная стратегия исследования
- Постепенное уменьшение параметра ε
- Визуализация прогресса обучения
- Сохранение обученной Q‑таблицы

## Результаты обучения

После 200 эпизодов агент достигает средней награды около 450, что соответствует успешному решению задачи.
