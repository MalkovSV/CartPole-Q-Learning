

# Использование проекта

## Запуск обучения

1. Перейдите в директорию проекта:

   ```bash
   cd CartPole-Q-Learning

 2. Запустите скрипт обучения:

    ```bash
    python src/cart_pole_q_learning.py 

## Параметры обучения
    В коде можно настроить следующие параметры:

    Параметр	Значение по умолчанию	Описание
    EPISODES	200	Количество эпизодов обучения
    LEARNING_RATE	0.1	Скорость обучения
    DISCOUNT	0.95	Коэффициент дисконтирования
    EPSILON	1.0	Начальное значение ε для ε‑жадной стратегии
    RENDER_EVERY	10	Частота отображения визуализации (каждый N‑й эпизод)

## Результаты
    После завершения обучения будут созданы:

    data/q_table.npy — обученная Q‑таблица;

    data/q_learning_training_progress.png — график прогресса обучения.
