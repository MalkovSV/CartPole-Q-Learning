

# Использование проекта

## Запуск обучения

1. Перейдите в директорию проекта:

   ```bash
   cd CartPole-Q-Learning

 2. Запустите скрипт обучения:

    ```bash
    python src/cart_pole_q_learning.py 

## Настройка параметров
    Измените значения в словаре CONFIG в файле src/q_learning_cart_pole.py:


    Параметр	            Описание	                        Значение по умолчанию

    EPISODES	            Количество обучающих эпизодов	        200
    RENDER_EVERY	        Визуализировать каждый N‑й эпизод	    10
    LEARNING_RATE	        Скорость обучения	                    0.1
    DISCOUNT	            Коэффициент дисконтирования	            0.95
    EPSILON	Начальное значение ε для ε‑жадной стратегии	            1.0
    START_EPSILON_DECAYING	Эпизод начала убывания ε	            1
    END_EPSILON_DECAYING	Эпизод окончания убывания ε	            100
    DISCRETE_OS_SIZE	    Размерность дискретизации 
                            (4 элемента для CartPole)	            [20, 20, 20, 20]
    TARGET_REWARD	        Целевая награда для ранней остановки	450
    SAVE_MODEL_EVERY	    Сохранять чекпоинт каждые N эпизодов	50

## Результаты

    После завершения обучения в папке data/ будут созданы:

   - q_table.npy — финальная Q‑таблица.

   - q_table_checkpoint_<номер>.npy — чекпоинты модели (если SAVE_MODEL_EVERY > 0).

   - training_log.csv — лог обучения с наградами и эпсилоном.

   - q_learning_training_progress.png — график прогресса обучения.

   - epsilon_decay.png — график изменения эпсилона.

   ## Анализ результатов

    В консоли выводится:

    - Статистика по каждому 10‑му эпизоду (награда, эпсилон).

    - Сообщение о достижении цели (если средняя награда ≥ TARGET_REWARD).

    - Итоговая статистика: максимальная, минимальная, средняя награда, стандартное отклонение.
